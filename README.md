# An-lise-de-Vendas---Base-de-Dados-Sales
Este projeto consiste em uma análise exploratória e agregada de uma base de dados de vendas, com o objetivo de identificar padrões, principais clientes, produtos mais vendidos, faturamento por produto e desempenho ao longo do tempo.
A base de dados utilizada contém informações de pedidos, incluindo:
- Número do pedido
- Data do pedido
- Código do produto
- Quantidade vendida
- Valor da venda
- Nome do cliente
- Região/território

## Objetivos da Análise

1. Identificar os produtos com maior volume de vendas e faturamento.
2. Determinar os clientes mais relevantes em termos de valor total de compras.
3. Analisar o faturamento agregado por território/região.
4. Avaliar a evolução das vendas ao longo dos anos.
5. Calcular métricas unitárias, como preço médio por unidade vendida.

## Principais Etapas

- Limpeza e preparação da base de dados:
  - Tratamento de valores nulos.
  - Conversão de datas para formato datetime.
  - Criação de colunas derivadas, como ano e mês.
- Agrupamento e agregação de dados:
  - Soma de vendas por produto, cliente e território.
  - Identificação das top vendas e top clientes.
- Cálculo de métricas adicionais:
  - Valor médio unitário por produto.
- Ordenação de resultados para destacar os principais insights.
- Visualização resumida para rápida interpretação.

## Tecnologias Utilizadas

- Python
- Pandas
- Jupyter Notebook

## Resultado Esperado

- Tabelas agregadas com os produtos mais vendidos, clientes com maior faturamento e evolução anual das vendas.
- Métricas derivadas que permitem tomar decisões estratégicas sobre produtos e clientes.
- Notebook organizado e comentado, pronto para visualização e compartilhamento.
